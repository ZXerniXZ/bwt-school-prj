
## Avvio del programma

### Prerequisiti
- Python 3
- Dipendenze installate: `pip install -r requirements.txt`

### Avvio
1. Avviare il server socket: `python server.py`
   - Il server ascolta sulla porta 65432

2. Avviare il client web: `python client.py`
   - Il client web è disponibile su http://127.0.0.1:5050

### Esecuzione test
Per eseguire tutti i test: `python run_all_tests.py`
